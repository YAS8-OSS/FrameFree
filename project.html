دليل المبتدئ لاستخدام GitHub مع الذكاء الاصطناعي

  1. إنشاء حساب على GitHub: قم بزيارة https://github.com/join وأنشئ حساب جديد بتفعيل بريدك الإلكتروني.
  2. تسجيل الدخول إلى GitHub: ادخل إلى https://github.com/login واستخدم بيانات حسابك لتسجيل الدخول.
  3. إنشاء مستودع جديد (Repository): اضغط زر New repository، أدخل اسم المستودع وحدد خصوصيته (عام أو خاص)، ثم اضغط Create repository.
  4. استخدام ChatGPT لمساعدتك في البرمجة: افتح https://chat.openai.com واطلب كتابة الأكواد أو شرحها حسب مشروعك.
  5. رفع الكود إلى GitHub: استخدم GitHub Desktop أو سطر الأوامر لإضافة الملفات، الالتزام (commit)، والرفع (push) إلى المستودع.
  6. أدوات مساعدة إضافية: - GitHub Copilot للمساعدة في كتابة الكود. - GitHub Desktop لتسهيل رفع الملفات.
